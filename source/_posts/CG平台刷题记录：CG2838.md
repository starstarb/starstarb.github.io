2838. 元音字母

【问题描述】

输入一个字符串，提取其中的元音字母(a、e、i、o、u，包括大小写)并输出。

【输入形式】

输入中包含多个测试数据。每个测试数据占一行，为一串字符，字符长度最长可达100个字符。输入的最后一行为end，表示输入结束。

【输出形式】

对输入中的每个字符串，提取其中的元音字母并输出。

【样例输入】

I love china.

end

【样例输出】

Ioeia
```
#include <iostream>
//#include <cstring>
using namespace std;
int main() {
	string s; 
   
  while(1) {
  	string att;
  	getline(cin,s);
  	if(s=="end"){
        	break;
           }
    for (int i=0;i<s.length();i++) {

        switch(s[i]){
        case 'a' :
		    att+="a";break;
		case 'e' :
		    att+="e";break;
		case 'i' :
		    att+="i";break;
		case 'o' :
		    att+="o";break;
		case 'u' :
		    att+="u";break;
		case 'A' :
		    att+="A";break;
		case 'E' :
		    att+="E";break;
		case 'I' :
		    att+="I";break;
		case 'O' :
		    att+="O";break;
		case 'U' :
		    att+="U";break;

        }
        
    }
    
    cout<<att<<endl;
  }
  return 0;
}
    

```
